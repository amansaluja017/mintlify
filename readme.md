# Mintlify Landing Page Recreation

This project is a recreation of the Mintlify landing page, showcasing modern web design with a focus on clean typography, elegant color schemes, and responsive layout.

## üìã Project Overview

This is a responsive landing page that highlights the key features and benefits of Mintlify, an intelligent knowledge platform designed to help teams create and maintain world-class documentation built for both humans and AI.

### Key Sections:
- **Navigation Bar** - Sticky header with logo, menu links, and action buttons
- **Hero Section** - Eye-catching banner with call-to-action
- **Feature Showcase** - Customer logos and feature cards
- **Services Section** - Detailed explanation of core features
- **Call to Action** - Email subscription form and additional resources

### Fonts Used:

1. **Inter** (`Inter-VariableFont_opsz,wght.ttf`)
   - Primary font family for body text and headings
   - Variable font allowing flexible weight adjustments
   - Location: `fonts/Inter-VariableFont_opsz,wght.ttf`
   - Usage: All text content, navigation, buttons
   - Font Weight: 500 (medium) - Default for body

2. **Geist Mono** (`GeistMono-VariableFont_wght.ttf`)
   - Monospace font for technical or code-related content
   - Location: `fonts/GeistMono-VariableFont_wght.ttf`
   - Fallback: system sans-serif fonts
